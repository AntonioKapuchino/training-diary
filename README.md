# Дневник тренировок

Личный дневник тренировок: выбираешь день, добавляешь упражнения с весами и подходами, пишешь комментарии про ощущения. Графики прогресса, каталог упражнений и шаблоны тренировок.

Дизайн — светлый минимал в стиле Apple Health. Всё хранится **локально на устройстве** (IndexedDB), без регистрации и интернета.

## Возможности

- 📅 **Дневник** — список тренировок по дням, объём и группы мышц на каждой карточке
- 🏋️ **Редактор тренировки** — упражнения, подходы (повторы × вес), комментарий к каждому упражнению и общая заметка к дню. Прошлые веса подставляются автоматически
- 📚 **Каталог упражнений** — 30+ готовых по группам мышц + можно добавлять свои
- 📈 **Прогресс** — графики роста макс. веса и объёма по каждому упражнению, рекорды (PR)
- 🔁 **Шаблоны** — сохранить тренировку как шаблон и применить в один тап

## Запуск (веб)

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/ (готовый PWA)
npm run preview  # просмотр собранной версии
```

Это **PWA** — на телефоне можно «Добавить на главный экран», работает офлайн как обычное приложение.

## Дальше: нативное приложение под Android и iOS

План — обернуть тот же код в [Capacitor](https://capacitorjs.com), который собирает нативные приложения под обе платформы из этой же веб-сборки:

```bash
npm install @capacitor/core @capacitor/cli
npx cap init "Дневник тренировок" com.example.diary --web-dir=dist
npm install @capacitor/android @capacitor/ios
npm run build
npx cap add android   # проект Android Studio → .apk / Google Play
npx cap add ios       # проект Xcode → App Store (нужен Mac)
npx cap sync
```

На этом этапе сгенерируем нативные иконки и сплеш-скрины под обе платформы.

## Стек

React 18 · TypeScript · Vite · React Router · Dexie (IndexedDB) · vite-plugin-pwa. Графики — собственные на SVG, без тяжёлых зависимостей.

## Структура

```
src/
  db/          схема, типы, функции работы с данными
  lib/         даты (русская локализация), цвета групп мышц
  components/  иконки, нижний лист (Sheet), выбор упражнения, график
  pages/       Diary, WorkoutEditor, Progress, Catalog, Templates
```
